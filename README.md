## 🥕 못생겼지만 맛있는 농산물을 구독해봐요!
 
상품 가치가 하락한 농산물을 구독하여 합리적 가격으로 판매하는 `반응형 웹 사이트` 제작 <br>
상품, 주문, 리뷰, 결제, 로그인, 마이페이지, 관리자 페이지를 `반응형`으로 구현하였다.
