# SpringTeamProject / 울퉁불퉁's

### 프로젝트 계기

>  충분히 맛있지만 못생겼거나 생산량이 많다는 이유만으로 버려지는 농산물을 구독 방식과 합리적 가격으로 판매하는 웹 페이지 제작


상품, 주문, 리뷰, 결제, 로그인, 마이페이지, 관리자 페이지를 구현하여 완성하였다.
